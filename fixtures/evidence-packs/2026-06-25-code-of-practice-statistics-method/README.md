# Code of Practice for Statistics Method Evidence Pack

This is the first controlled real/sandbox Evidence Pack.

It uses one real public source:

- `https://code.statisticsauthority.gov.uk/`

The pack tests whether Evidence Pack v1 can preserve a small proof trail around a public source.

It is not a TWIS article pack.

It does not make a wider claim about whether public statistics are always trustworthy, high quality, or valuable in practice.

## Research question

What does the official Code of Practice for Statistics homepage say about the purpose and benefit of the Code?

## Scope

In scope:

- one official public webpage.
- one narrow source-description claim.
- one evidence item.
- basic source authority mapping.
- basic timeline, denial-check, and human-review records.

Out of scope:

- judging the whole UK statistical system.
- checking whether every producer follows the Code.
- making a TWIS article from this source.
- making any claim about government behaviour beyond what this source says.

## Good enough

This pack is good enough when it validates and clearly separates:

- what the source says.
- what the pack proves.
- what the pack does not prove.
