# Publication Gap Checklist

## Current publication gate

Status: not ready for publication.

This checklist records what is still needed before the China TWIS article can be treated as publication-ready rather than article-development material.

## Gate 1: article claim calibration

- [ ] Rewrite the article so its claims match the current evidence level.
- [ ] Use cautious language where the pack only supports context or risk.
- [ ] Avoid saying the pack proves measured harm from the EU parcel duty.
- [ ] Avoid saying all tariff or customs-duty costs are passed directly to consumers.
- [ ] Avoid saying poorer UK consumers depend on Shein, Temu, AliExpress, or similar platforms unless a direct source is added.

Current status: open.

## Gate 2: primary policy source

- [ ] Add a primary Council or official EU source for the temporary duty decision and timetable.
- [ ] Confirm the amount, start date, scope, and legal/policy route.
- [ ] Separate the Reuters-reported duty point from the European Commission customs-reform context.

Current status: partly supported, still open.

## Gate 3: direct platform-use evidence

- [ ] Add evidence on who uses low-value direct-from-China platforms.
- [ ] Prefer UK or EU evidence broken down by income, age, or household constraint.
- [ ] Separate general bargain-seeking behaviour from dependence or necessity.
- [ ] Avoid using US income evidence as if it proves UK low-income platform reliance.

Current status: context only, still open.

## Gate 4: customs-duty pass-through evidence

- [ ] Add evidence on pass-through for low-value parcel duties specifically.
- [ ] Add evidence on whether platforms, sellers, logistics firms, or consumers absorb the added cost.
- [ ] Add platform-specific evidence if available.
- [ ] Distinguish general tariff pass-through from this specific EU low-value parcel duty.

Current status: context only, still open.

## Gate 5: consumer-impact evidence

- [ ] Add evidence that low-income or financially constrained households use the affected goods or platforms.
- [ ] Add evidence that any added cost would be material for those households.
- [ ] Separate general cheap-goods inflation from direct parcel-duty effects.
- [ ] Check whether affected consumers are in the UK, EU, or both.

Current status: context only, still open.

## Gate 6: historic outsourcing evidence

- [ ] Add sources on Western corporate outsourcing to China.
- [ ] Add sources on margin, wage, supply-chain, or cost incentives.
- [ ] Add sources on policy choices that enabled the system.
- [ ] Avoid reducing a multi-decade economic process to a single unsupported motive claim.

Current status: unsupported beyond article framing.

## Gate 7: contradiction and counter-source checks

- [ ] Search for evidence that the duty may not be passed to consumers.
- [ ] Search for evidence that the measure is aimed mainly at safety, compliance, tax fairness, or market regulation rather than consumer punishment.
- [ ] Search for evidence that poorer consumers are not the main users of affected platforms.
- [ ] Record any contradiction or weakening evidence in the negative evidence log.

Current status: not done.

## Gate 8: human editorial review

- [ ] Human reviewer checks all source records.
- [ ] Human reviewer checks every publication claim against a source or a clearly marked inference.
- [ ] Human reviewer checks article tone for overclaiming.
- [ ] Human reviewer confirms the article is either caveated enough to publish or still held back.

Current status: not done.

## Gate 9: publication decision

The article can move toward publication only when one of these is true:

- the article is rewritten as a cautious, clearly caveated analysis using only supported claims; or
- the missing direct evidence is added for platform use, pass-through, consumer impact, and historic outsourcing.

Until then, the safe decision remains:

Do not publish as fully evidenced.
