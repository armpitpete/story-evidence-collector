# Example Evidence Pack Fixture

This is a dummy evidence pack.

It exists to show how an evidence pack folder fits together.

It is not real political evidence.

## Pack purpose

This fixture tests the file layout introduced by Evidence Pack Assembly v1 and the manifest fields defined by Evidence Pack Manifest Schema v1.

## Safe use

Use this fixture for:

- schema validation
- folder layout checks
- future pack validator tests
- documentation examples

Do not use this fixture for:

- publication
- real claims
- real source authority decisions
- real contradiction findings
